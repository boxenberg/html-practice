How do I see the differences between HTML and CSS?

  1. What is the difference between the usage of HTML and CSS?

    HTML is for giving a frame to the webpage and showing the content, CSS is for styling, making it look better and for instance adapt to different devices.

  5. What is the difference between semantic and non-semantic elements?

    Semantic elements have a clearly defined function, non-semantic ones are used more for structuring or styling purposes and tell nothing about their contents.

  6. Why do we prefer using semantic elements?

    To better optimize the page for search engines, also to have a clearer structure for self or other developers.

  7. Do all browsers support all of the newest HTML / CSS elements?

    Not fully on all devices, however most major browser support all the most important elements
